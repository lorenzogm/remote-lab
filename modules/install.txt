Installation:


Add these css lines to your theme css file:

.calendar_box {
  width: 100%;
  margin-bottom: 18px;
  padding: 0;
  border-collapse: separate;
  font-size: 13px;
}

.calendar_box th, .calendar_box td {
  padding: 10px 10px 9px;
  line-height: 13.5px;
  text-align: left;
  vertical-align: middle;
  border-bottom: 1px solid #ddd;
}

.calendar_box th {
  padding-top: 9px;
  font-weight: bold;
  border-bottom-width: 2px;
}

.calendar_box .today {
    background: #E9EFF7;
} 

You can style calendar as you want, this is just default.

Calendar now has its item types.
When creating new item type you can use template for this item. In template you can use all pyroCMS tags, and tags for calendar entry:
	{{ id }} - calendar entry id
	{{ date_start }} - start date of event
	{{ date_end }} - end date of event
	{{ restricted_to }} - item preview restrictions in csv
	{{ category }} - category id
	{{ item_type }} - item type id
	{{ recurrence }} - event recurrence
	{{ stream_entry_id }} - entry id in used stream
	{{ created_on }} - created date
	{{ updated_on }} - update date
	{{ created_by }} - creator user id
also you can use every field in your item type, you can access it in this syntax {{ entry:field_slug }}, it will print stream entry id,
	{{ entry:title }} - it will print title of event and etc.

When adding new fields to item type, click button in shortcuts bar for this action!
